# Dormatrix – Unified Hostel Operations Platform

## Problem Statement
Hostel wardens manage attendance, gate passes, visitor logs, and mess feedback through registers and WhatsApp groups, which creates operational chaos and poor record management. There is no centralized system to manage hostel operations efficiently.

## Solution
Dormatrix is a web-based Unified Hostel Operations Platform that digitizes hostel operations such as:
- Room allocation
- Digital gate pass approvals
- Parent SMS notifications
- Complaint management with SLA tracking
- Mess menu voting
- Visitor log system
- Notice board communication
- Dashboard analytics

The system provides a centralized platform for both administrators and tenants.
